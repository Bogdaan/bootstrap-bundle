Bootstrap Collection

Bootstrap collections are an extension of Symfonys default collection form type and add the JavaScript to make the form dynamic. They can be nested.

Warning The prototype_name of the subcollection must be different from the name of the parent collection.

$bcCollectionForm = $this->createFormBuilder([])
    ->add('subcol', 'bootstrap_collection', array(
        'type'               => 'bootstrap_collection',
        'allow_add'          => true,
        'allow_delete'       => true,
        'add_button_text'    => 'Add Level 1',
        'delete_button_text' => 'Delete Level 1',
        'sub_widget_col'     => 9,
        'button_col'         => 3,
        'options'            => array(
            'type'               => 'text',
            'allow_add'          => true,
            'allow_delete'       => true,
            'add_button_text'    => 'Add Level 2',
            'delete_button_text' => 'Delete Level 2',
            'prototype_name'     => '__subname__',
            'sub_widget_col'     => 8,
            'button_col'         => 4
        )
    ))
    ->setData(array(
        'subcol' => array(
            array('Answer 1a', 'Answer 1b'),
            array('Answer 2a', 'Answer 2b')
        )
    ))
    ->getForm();

Side-by-side Forms

You can mix the HTML and CSS structures provided by Bootstrap anytime with widgets and extensions provided by BootstrapBundle. For example, you can display two widgets side-by-side.

<div class="row">
    <div class="col-lg-6">
        {{ form_row(sidebysideForm.firstName, { 'style': 'horizontal', 'label_col': 4, 'widget_col': 8 }) }}
    </div>
    <div class="col-lg-6">
        {{ form_row(sidebysideForm.lastName, { 'style': 'horizontal', 'label_col': 4, 'widget_col': 8 }) }}
    </div>
</div>
